# Getting Started with Chirpy

Chirpy is a minimal, responsive, and feature-rich Jekyll theme for technical writing. This post will guide you through the basic setup and configuration of the theme.

## Features

- Dark/Light mode
- Localization
- Search
- Table of Contents
- Comment Systems
- Mathematics
- Mermaid Diagrams
- Preview Images

## Basic Configuration

The basic configuration is done through `_config.yml`. Here are the key settings you need to configure:

1. Site Information
2. Author Details
3. Social Links
4. Comments System
5. Analytics

## Writing Posts

Creating new posts is simple. Just create a new markdown file in the `_posts` directory with the format:

```
YYYY-MM-DD-title.md
```

Make sure to include proper front matter at the top of each post:

```yaml
---
title: Your Post Title
date: YYYY-MM-DD HH:MM:SS +/-TTTT
categories: [Main Category, Sub Category]
tags: [tag1, tag2]
---
```
